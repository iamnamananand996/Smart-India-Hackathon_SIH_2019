Pest Prediction

The pest prediction model is developed using Tensorflow library. The image visionary is analyzed using convolutional neural network (CNN). The model can detect the pest on the uploaded image and give measures to be taken.

Weather Prediction

The weather prediction is given using OpenWeatherMap API. This functionality helps the farmer take decisions considering weather's effect on farming.

Crop Prediction

Based on the region selected, the model can give crop prediction based on the soil's Nitrogen(N), Phosporous (P), Potassium (K) and pH values. It is developed using K-Neighbours Classifier algorithm. It can also give crop prediction for manually entered NPK and pH values.
